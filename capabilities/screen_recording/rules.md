# Screen Recording Rules

## Cursor Styles

| Style | Shape | Use Case |
|-------|-------|----------|
| Default | Arrow | General navigation |
| Pointer | Hand | Clickable elements |
| Text | I-beam | Text selection |
| Crosshair | + | Precision targeting |
| Custom Branded | Logo | Brand consistency |

## Click Indicators

| Indicator | Style | Duration |
|-----------|-------|----------|
| Ripple | Expanding circle | 0.5s |
| Highlight | Colored circle | 0.3s |
| Pulse | Flash effect | 0.2s |
| Sound | Click audio | Instant |

## Zoom Rules

| Target | Zoom Level | Use Case |
|--------|------------|----------|
| Full screen | 100% | Context |
| Panel | 125% | Section focus |
| Element | 150% | Detail |
| Text | 200% | Reading |

## Annotation Types

| Type | Style | Use Case |
|------|-------|----------|
| Callout | Box with arrow | Explanation |
| Tooltip | Small popup | Hint |
| Highlight | Colored overlay | Emphasis |
| Number | Circled number | Steps |
| Timer | Countdown | Wait periods |

## Transition Rules

| Transition | Duration | Use Case |
|------------|----------|----------|
| Cut | 0s | Quick change |
| Cross-fade | 0.3s | Smooth transition |
| Slide | 0.4s | Navigation |
| Zoom | 0.5s | Focus change |

## Video Specs

| Spec | Value | Reason |
|------|-------|--------|
| Resolution | 1920x1080 | Standard HD |
| FPS | 30 | Smooth motion |
| Bitrate | 8 Mbps | Quality |
| Format | MP4 (H.264) | Compatibility |

## Cursor Animation

| Movement | Style | Speed |
|----------|-------|-------|
| Idle | Slight float | 2s loop |
| Moving | Smooth curve | 0.5s/100px |
| Clicking | Scale bounce | 0.2s |
| Hovering | Gentle pulse | 1s loop |

## Prompt Template

```
Screen recording of [application]:
- Action: [user flow]

```
