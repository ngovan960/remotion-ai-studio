# YouTube Retention Skill

> Tối ưu retention cho YouTube

## Retention Timeline

```
0-5s:    HOOK          → Pattern interrupt, shock
5-30s:   SETUP         → Promise, what viewer gets
30s-2m:  CONTEXT       → Build world, characters
2-4m:    CONFLICT      → Rising tension
4-6m:    MIDPOINT      → Twist, re-engage
6-8m:    ESCALATION    → Highest stakes
8-9m:    CLIMAX        → Peak emotion
9-10m:   RESOLUTION    → Payoff + CTA
```

## Retention Techniques

### 1. Pattern Interrupt (0-5s)
```
❌ "Hello everyone, today we're going to talk about..."
✅ "This man lost $40 million in 3 days."
```

### 2. Curiosity Gap
```
Promise: "By the end of this video, you'll understand why..."
Delay: Don't answer immediately
Partial: Give some info, hold back more
```

### 3. Open Loop
```
Start a story → Interrupt with another → Return later
"Alice faced a decision. But first, let's talk about Bob..."
```

### 4. Cliffhanger (every 2-3 min)
```
End section with: "And that's when everything changed..."
Or: "But the truth was even more shocking..."
```

### 5. Payoff
```
Deliver on promise from hook
Reference beginning: "Remember the $40M? Here's what happened..."
```

### 6. Rehook (mid-video)
```
New curiosity gap at midpoint
"But there's something I haven't told you yet..."
```

### 7. CTA (end)
```
"Subscribe for more stories like this"
"Watch this next video about..."
```

## Retention Checkpoints

- [ ] Hook strong in first 3 seconds?
- [ ] Promise made by 30 seconds?
- [ ] Curiosity loop every 30-60s?
- [ ] Cliffhanger before each major section?
- [ ] Midpoint twist at 40-60%?
- [ ] Payoff at end?
- [ ] CTA at very end?
