# Visual Prompt Engineering Skill (v2.0)

> 15-element prompt structure cho Flux/SDXL

## Prompt Template (15 Elements)

```
Subject + Environment + Lighting + Lens + Camera Angle + Composition + 
Depth of Field + Texture + Foreground + Background + Mood + Color Palette + 
Film Grain + Rendering Style + Quality + Negative
```

## Ví Dụ Chi Tiết

```yaml
image_prompt:
  Subject: "12 km asteroid entering Earth atmosphere, rocky texture, fire trail"
  Environment: "deep outer space, stars background, cosmic dust"
  Lighting: "orange plasma glow, dramatic rim light, volumetric"
  Lens: "35mm"
  Camera Angle: "slightly low, looking up"
  Composition: "rule of thirds, subject at left third"
  Depth of Field: "deep, everything sharp"
  Texture: "rough asteroid surface, fire particles"
  Foreground: "debris particles floating"
  Background: "stars, distant galaxies"
  Mood: "catastrophic, impending doom"
  Color Palette: "orange, red, deep blue space, black"
  Film Grain: "Kodak Vision3 500T, subtle"
  Rendering Style: "ultra realism, BBC documentary"
  Quality: "ultra detailed, 8k, photorealistic"
  Negative: "logo, watermark, low quality, blurry, text, anime"
```

## Atmosphere Keywords

| Type | Keywords |
|------|----------|
| Fog | "volumetric fog, mist, haze" |
| Dust | "dust particles, floating debris" |
| Ash | "ash falling, volcanic particles" |
| Smoke | "smoke rising, haze" |
| Light rays | "volumetric light, god rays, light beams" |

## Texture Keywords

| Type | Keywords |
|------|----------|
| Skin | "detailed skin texture, pores, wrinkles" |
| Rock | "rough stone surface, cracks, mineral veins" |
| Metal | "brushed metal, reflections, scratches" |
| Fabric | "cloth texture, weave pattern, folds" |

## Color Grade Keywords

| Style | Keywords |
|-------|----------|
| Teal/Orange | "teal shadows, orange highlights" |
| Desaturated | "muted colors, low saturation" |
| High Contrast | "deep blacks, bright whites" |
| Vintage | "sepia tones, film grain, faded" |
