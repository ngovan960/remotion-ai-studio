# Visual Thinking Skill

> Chuyển ý tưởng thành hình ảnh cụ thể

## Core Rule

After every sentence, ask:
**"What will the viewer SEE on screen?"**

If answer is "just text" → rewrite.

## Abstract → Concrete

| Abstract | Concrete Visual |
|----------|-----------------|
| AI | Neural network nodes glowing |
| Time | Clock counting down |
| Connection | Two hands touching, sparks |
| Growth | Plant growing, chart rising |
| Breakthrough | Door opening to light |
| Failure | Broken glass, falling |
| Success | Trophy, celebration |
| Knowledge | Books opening, light |

## B-roll Rule

Every 15-20 seconds → Insert 1 B-roll

## "Without Sound" Test

- [ ] Viewer understands without audio?
- [ ] Every scene has ONE clear visual?
- [ ] No scene is just text?
- [ ] Visual hierarchy clear?

## Scene Length

One scene = 5-10s = ONE idea

| Content | Scenes Needed |
|---------|---------------|
| Single idea | 1 |
| Two ideas | 2 |
| Complex concept | 3-4 |
| Full explanation | 5+ |
