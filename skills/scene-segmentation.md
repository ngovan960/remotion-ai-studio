# Scene Segmentation Skill

> Chia video thành scenes và shots

## Rule: One Scene = One Idea

## Scene Types

| Type | Duration | Purpose |
|------|----------|---------|
| Hook | 3-5s | Grab attention |
| Animation | 5-10s | Show concept |
| B-roll | 3-5s | Context/mood |
| Infographic | 5-8s | Data/stats |
| Zoom | 3-5s | Emphasize detail |
| Talking Head | 8-15s | Direct address |
| Comparison | 8-12s | Before/after |
| Demo | 10-15s | Show process |
| Conclusion | 5-10s | Wrap up |

## Multi-Act Structure

```
ACT 1: Hook (0-30s)
ACT 2: Setup (30s-2min)
ACT 3: Conflict (2-5min)
ACT 4: Midpoint (4-6min)
ACT 5: Escalation (6-8min)
ACT 6: Climax (8-9min)
ACT 7: Resolution (9-10min+)
```

## Transitions

| Type | Example |
|------|---------|
| Delayed answer | "But the real reason..." |
| Cliffhanger | "And then everything changed..." |
| Question hook | "Why didn't anyone tell?" |
| Contrast | "They thought X. Reality was Y." |
| Foreshadow | "If only they knew..." |
