# Color Theory Skill

> Lý thuyết màu sắc cho video AI

## Color Psychology

| Color | Emotion | Association | Use |
|-------|---------|-------------|-----|
| **Red** | Danger, passion, energy | Love, war, urgency | Action, climax |
| **Blue** | Trust, calm, sadness | Tech, corporate, ocean | Tech, mystery |
| **Green** | Nature, growth, health | Environment, money | Nature, positive |
| **Yellow** | Optimism, warmth, caution | Sunshine, warning | Hope, attention |
| **Orange** | Energy, enthusiasm | Autumn, creativity | Warmth, action |
| **Purple** | Mystery, luxury, wisdom | Royalty, magic | Fantasy, premium |
| **Black** | Power, elegance, death | Formal, mystery | Horror, luxury |
| **White** | Purity, clean, empty | Medical, minimal | Clean, sterile |

## Color Harmony

| Type | Colors | Effect |
|------|--------|--------|
| **Complementary** | Blue + Orange | High contrast |
| **Analogous** | Blue + Green + Teal | Harmonious |
| **Triadic** | Red + Yellow + Blue | Vibrant |
| **Monochromatic** | Shades of one | Elegant |

## Color Script Template

```markdown
## COLOR SCRIPT: [Video Name]

| Act | Time | Palette | Mood | Hex Codes |
|-----|------|---------|------|-----------|
| Hook | 0-30s | Cool blue | Mystery | #1A1A2E, #16213E |
| Setup | 30s-2min | Neutral | Calm | #F5F5F5, #333333 |
| Conflict | 2-5min | Warm orange | Tension | #FF6B35, #F7C59F |
| Climax | 5-8min | Dark red | Danger | #8B0000, #2D0000 |
| Resolution | 8-10min | Soft blue | Peace | #87CEEB, #FFFFFF |
```

## Color in Prompts

```
Color palette: [primary], [secondary], [accent]
Lighting color: [warm/cool/neutral] tones
Mood color: [emotion] color grading
```

Examples:
- `Cool blue tones, mysterious atmosphere`
- `Warm golden hour lighting, amber accents`
- `Dark desaturated palette, noir style`
