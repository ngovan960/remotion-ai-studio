## Prompt Template for Metaphors

```
Visual metaphor for [concept]: [concrete visual description],
symbolic representation, cinematic lighting
```

Examples:
- `Visual metaphor for "breakthrough": door opening to blinding light, dramatic`
- `Visual metaphor for "connection": two hands reaching toward each other, sparks`
- `Visual metaphor for "growth": small plant growing into tree, time-lapse style`
