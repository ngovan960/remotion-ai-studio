# Research Skill

> Nghiên cứu chủ đề, fact checking, timeline, source ranking

## Research Pipeline

```
Topic
  ↓
Web Search (Agent Reach)
  ↓
Source Evaluation
  ↓
Fact Verification
  ↓
Timeline Construction
  ↓
Knowledge Summary
  ↓
Story
```

## Source Ranking

| Priority | Source Type | Reliability |
|----------|-------------|-------------|
| 1 | Official docs, papers | ⭐⭐⭐⭐⭐ |
| 2 | Government data | ⭐⭐⭐⭐⭐ |
| 3 | News outlets | ⭐⭐⭐⭐ |
| 4 | Expert interviews | ⭐⭐⭐⭐ |
| 5 | Forums, discussions | ⭐⭐⭐ |
| 6 | Social media | ⭐⭐ |

## Fact Verification Rules

1. Cross-reference 2+ sources
2. Find contradictions
3. Flag uncertain information
4. Never present opinion as fact
5. Cite sources for all claims

## Timeline Construction

```
Event A (Date) → Event B (Date) → Event C (Date)
     ↓                  ↓                ↓
   Cause              Effect            Result
```

## Knowledge Summary Template

```markdown
## Research Summary: [Topic]

### Key Facts
- Fact 1 (source)
- Fact 2 (source)

### Timeline
- [Date]: Event 1
- [Date]: Event 2

### Key People
- Person A: Role, importance
- Person B: Role, importance

### Conflicting Information
- Claim A vs Claim B (sources)

### Sources
- [URL 1] - [Summary]
- [URL 2] - [Summary]
```

## Quality Check

- [ ] At least 3 sources per major claim?
- [ ] Timeline chronological?
- [ ] No unverified facts?
- [ ] All sources cited?
