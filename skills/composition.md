## Rule of Thirds

```
┌───────┬───────┬───────┐
│       │       │       │
│   ●───┼───────┼───●   │  ← Place subjects at intersections
│       │       │       │
├───────┼───────┼───────┤
│       │       │       │
│       │       │       │
├───────┼───────┼───────┤
│       │       │       │
│   ●───┼───────┼───●   │
│       │       │       │
└───────┴───────┴───────┘
```

## Leading Lines

```
Road/path    → leads eye into image
Fence/rail   → creates depth
River/stream → guides movement
Architecture → frames subject
```

## Symmetry

| Type | When | Example |
|------|------|---------|
| Mirror | Formal, powerful | Architecture, reflections |
| Approximate | Natural, organic | Landscapes, crowds |
| Radial | Center focus | Wheels, flowers, eyes |

## Framing Techniques

| Technique | Description | Effect |
|-----------|-------------|--------|
| Frame within frame | Door, window, arch | Depth, focus |
| Natural frame | Trees, branches | Context |
| Negative space | Empty area | Emphasis, isolation |

## Depth Layers

```
Foreground  → Subject, props, detail
Midground   → Supporting elements
Background  → Environment, context
```

## Composition Prompt Template

```
[Subject] placed at rule of thirds intersection,
[Leading lines] guiding viewer's eye,
[Frame within frame] technique,
Deep/Moderate/Shallow depth of field
```
