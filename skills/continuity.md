# Continuity Skill

> Đảm bảo nhất quán xuyên suốt video

## Character Continuity

| Attribute | Rule |
|-----------|------|
| Face | NEVER change |
| Hair | NEVER change |
| Clothing | NEVER change mid-scene |
| Accessories | NEVER change |
| Voice | SAME throughout |

## Environment Continuity

| Rule | Example |
|------|---------|
| Same location | Lab → Lab ✅ |
| Different location | Lab → Desert ❌ (need transition) |
| Time change | Day → Sunset → Night ✅ (gradual) |

## Style Continuity

| Rule | Example |
|------|---------|
| Same style | Realistic → Realistic ✅ |
| Different style | Realistic → Anime ❌ |

## Color Continuity

```markdown
| Scene Range | Palette | Mood |
|-------------|---------|------|
| 1-10 | Blue | Mystery |
| 11-20 | Gold | Hope |
| 21-30 | Red | Danger |
| 31-40 | Soft blue | Peace |
```

## Camera Continuity

Smooth transitions only:
```
Wide → Medium → Close-up → Wide ✅
Wide → Close-up → Wide → Medium ❌
```

## 180-Degree Rule

- Pick ONE side for camera
- Keep camera on same side
- Switch sides only with clear reason

## Quality Check

- [ ] Character consistent?
- [ ] Environment consistent?
- [ ] Color palette maintained?
- [ ] Camera smooth?
- [ ] Timeline logical?
