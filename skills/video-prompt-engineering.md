# Video Prompt Engineering Skill (v2.0)

> 5 motion layers cho Wan 2.2

## 5 Motion Layers

```yaml
video_prompt:
  Subject Motion: "asteroid moving toward camera, rotating slowly"
  Camera Motion: "slow push in from space, 30 degree angle"
  Secondary Motion: "debris particles floating around asteroid"
  Environmental Motion: "stars twinkling, cosmic dust drifting"
  Object Interaction: "fire trail expanding as asteroid enters atmosphere"
  Duration: "6 seconds"
  FPS: 24
  Resolution: "720p"
```

## Motion Types

| Layer | Mô tả | Ví dụ |
|-------|-------|-------|
| **Subject Motion** | Chuyển động chính | "walking", "flying", "exploding" |
| **Camera Motion** | Chuyển động camera | "push in", "pan left", "orbit" |
| **Secondary Motion** | Chuyển động phụ | "debris flying", "smoke rising" |
| **Environmental Motion** | Môi trường | "wind blowing", "dust floating" |
| **Object Interaction** | Tương tác | "touching", "breaking", "spreading" |

## Camera Motions

| Motion | Prompt |
|--------|--------|
| Push in | "camera slowly pushing forward" |
| Pull out | "camera slowly pulling back" |
| Pan left | "camera panning left smoothly" |
| Pan right | "camera panning right smoothly" |
| Orbit | "camera orbiting around subject" |
| Tracking | "camera tracking from side" |
| Crane up | "camera rising, crane shot" |
| Handheld | "handheld camera, slight shake" |

## Duration Rules

| Content | Duration |
|---------|----------|
| Single action | 3-5s |
| Transition | 2-3s |
| Establishing | 5-8s |
| Detail reveal | 4-6s |
| Emotional beat | 4-6s |
