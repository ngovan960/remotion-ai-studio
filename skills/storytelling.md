# Storytelling Skill

> Kể chuyện cho documentary video

## Story Structures

### Hero's Journey
```
Ordinary World → Call to Adventure → Refusal → Mentor
  → Crossing Threshold → Tests → Ordeal → Reward
  → Road Back → Resurrection → Return with Elixir
```

### Three Act Structure
```
Act 1: Setup (25%)
  → Introduce world, characters, conflict

Act 2: Confrontation (50%)
  → Rising action, obstacles, midpoint twist

Act 3: Resolution (25%)
  → Climax, falling action, resolution
```

### Seven Act Structure (Documentary)
```
Act 1: Hook (0-30s)         → Shock fact
Act 2: Setup (30s-2min)     → World, characters
Act 3: Conflict (2-5min)    → Tension, obstacles
Act 4: Midpoint (4-6min)    → Plot twist
Act 5: Escalation (6-8min)  → Highest stakes
Act 6: Climax (8-9min)      → Peak emotion
Act 7: Resolution (9-10min+) → Payoff + insight
```

## Storytelling Techniques

| Technique | Description | Example |
|-----------|-------------|---------|
| **Cliffhanger** | End with suspense | "But then, everything changed..." |
| **Foreshadow** | Hint at future | "If only they knew what was coming..." |
| **Callback** | Reference earlier moment | "Just like in scene 1..." |
| **Contrast** | Before/after | "They thought X. Reality was Y." |
| **Reveal** | New information | "But the truth was..." |
| **Payoff** | Deliver on promise | "And that's why..." |

## Curiosity Loops

```
Question → Delay → Partial Answer → More Questions → Full Answer
```

Example:
- Q: "What happened next?"
- Delay: "But first, let's go back..."
- Partial: "It started with a simple email..."
- More: "And that email changed everything..."
- Answer: (later in video)

## Narrative Voice

| Style | When | Example |
|-------|------|---------|
| First person | Personal story | "I remember that day..." |
| Third person | Documentary | "He walked into the room..." |
| Omniscient | Epic | "Little did they know..." |
| Journalistic | News | "According to reports..." |
