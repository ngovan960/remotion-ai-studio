# B-roll Skill

> Lên kế hoạch B-roll cho video

## B-roll Types & Uses

| Type | When | Example |
|------|------|---------|
| **Establishing** | Opening scene | Skyline, landscape |
| **Transition** | Between sections | Blur, fade, movement |
| **Context** | Supporting narration | Relevant objects, places |
| **Emotion** | Reflective moments | Nature, people |
| **Time Pass** | Showing time passing | Clock, seasons, sunset |
| **Detail** | Emphasizing point | Close-up of object |
| **Abstract** | Emphasis, mood | Particles, light leaks |

## B-roll by Topic

### History
- Old books, maps, letters
- Museum artifacts, statues
- Historical photos (sepia)
- Architecture, monuments

### Technology
- Server rooms, circuits
- Code on screens
- Robots, AI interfaces
- Data visualizations

### Nature
- Mountains, ocean, forest
- Weather patterns
- Animals, plants
- Seasons changing

### Business
- Office environments
- Handshakes, meetings
- Charts, graphs
- City skylines

## B-roll Timing Rule

```
Every 15-20 seconds → Insert 1 B-roll

Scene (10s) → B-roll (3s) → Scene (10s) → B-roll (3s)
```

## Prompt Template

```
B-roll: [type], [subject], [time of day], [mood], 
cinematic, stock footage style
```

Example:
- `B-roll: establishing, city skyline at sunset, warm golden light, peaceful`
- `B-roll: detail, old book pages turning, candlelight, mysterious`
