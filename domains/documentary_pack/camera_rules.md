# Documentary Camera Rules

## Camera Types

| Type | Shot | Use |
|------|------|-----|
| **Establishing** | Wide | Opening scene, location |
| **Master** | Wide-Medium | Main action |
| **Close-up** | Close | Emotion, detail |
| **Insert** | Extreme close | Objects, text |
| **OTS** | Over-shoulder | Conversation |
| **POV** | First person | Immersive |
| **Aerial** | Drone | Scale, landscape |

## Camera Patterns

### Scene Flow
```
Establishing (Wide) → Master (Medium) → Close-up → Insert → Master → Establishing
```

### Transitions Between Shots
| From | To | Transition |
|------|-----|------------|
| Wide | Medium | Cut |
| Medium | Close-up | Cut |
| Close-up | Wide | Dissolve |
| Scene A | Scene B | Crossfade |

## Movement Rules

| Movement | Speed | Use |
|----------|-------|-----|
| Static | - | Data, charts |
| Slow zoom | 2-5s | Emphasis |
| Pan left/right | 3-5s | Reveal |
| Dolly in | 3-5s | Intimacy |
| Dolly out | 3-5s | Context |
| Tracking | Variable | Follow action |

## Camera by Scene Phase

| Phase | Camera | Movement |
|-------|--------|----------|
| Hook | Close-up → Wide | Slow reveal |
| Setup | Wide → Medium | Establishing |
| Conflict | Medium → Close-up | Tightening |
| Climax | Close-up | Static/intense |
| Resolution | Wide | Pull back |

## 180-Degree Rule

- Pick ONE side for camera
- Keep camera on same side
- Switch only with clear reason

## Quality Check

- [ ] Camera angles varied?
- [ ] Movements smooth?
- [ ] 180-degree rule followed?
- [ ] Transitions appropriate?
