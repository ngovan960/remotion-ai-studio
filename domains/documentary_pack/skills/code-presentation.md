# Code Presentation Skill

> Hiển thị code đẹp trong video

## Code Styles

| Style | Mô tả | Dùng khi |
|-------|-------|----------|
| **Syntax Highlight** | Màu theo language | Code snippet |
| **Focus Line** | Highlight dòng đang nói | Tutorial |
| **Scroll** | Cuộn code dài | Review code |
| **Zoom** | Phóng to chi tiết | Function detail |
| **Terminal** | Command line | Demo commands |

## Syntax Colors

| Element | Color | Ví dụ |
|---------|-------|-------|
| Keyword | Blue | `def`, `class`, `if` |
| String | Green | `"hello"` |
| Number | Orange | `42`, `3.14` |
| Comment | Gray | `# comment` |
| Function | Yellow | `def my_function()` |
| Variable | White | `x = 10` |
