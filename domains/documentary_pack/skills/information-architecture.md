# Information Architecture Skill

> Tổ chức thông tin: sections, headings, summary, flow logic

## Cấu Trúc Section

```markdown
# [Title] (H1)

## Tóm tắt (H2)
[1-2 câu tóm tắt toàn bộ]

## Nội dung (H2)
### Phần 1 (H3)
[Chi tiết]

### Phần 2 (H3)
[Chi tiết]

## Kết luận (H2)
[Tóm tắt lại]

## Tham khảo (H2)
[Nguồn]
```

## Flow Logic

```
Hook → Context → Main Content → Summary → Call to Action
```

## Heading Rules

| Level | Dùng khi | Ví dụ |
|-------|----------|-------|
| H1 | Title chính | "Lịch sử AI" |
| H2 | Section lớn | "Giai đoạn phát triển" |
| H3 | Sub-section | "Những năm 1950" |
| H4 | Chi tiết | "Alan Turing và AI" |

## Documentary Structure

```markdown
# [Title]

## Tóm tắt
[30 giây đọc]

## Bối cảnh
[Setup thế giới]

## Nội dung chính
### Phần 1
### Phần 2
### Phần 3

## Phân tích
[Đánh giá, so sánh]

## Kết luận
[Tóm tắt + ý nghĩa]

## Tham khảo
[Nguồn]
```

## Prompt Template

```
Information architecture: organize [topic] with clear hierarchy,
H1 title, H2 sections, H3 subsections,
logical flow from context to analysis to conclusion
```
