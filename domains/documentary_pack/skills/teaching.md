# Teaching Skill

> Giải thích từ dễ đến khó, chia nhỏ kiến thức

## Nguyên Tắc

| Nguyên tắc | Mô tả |
|------------|-------|
| **Từ dễ → khó** | Bắt đầu từ cái đã biết |
| **Chia nhỏ** | Mỗi concept một phần |
| **Analogies** | So sánh với cái quen thuộc |
| **Ví dụ** | Luôn có example |
| **Lặp lại** | Nhắc lại key points |

## Cấu Trúc Giải Thích

```
1. Hook: Câu hỏi hoặc ví dụ quen thuộc
2. Cơ bản: Giải thích đơn giản nhất
3. Chi tiết: Đi sâu hơn
4. Ví dụ: minh họa cụ thể
5. Tóm tắt: Nhắc lại key points
6. Exercise: Câu hỏi hoặc bài tập
```

## Analogies Cho Documentary

| Concept | Analogies |
|---------|-----------|
| Neural Network | "Như bộ não con người, có nhiều lớp tế bào thần kinh" |
| Algorithm | "Như công thức nấu ăn, có từng bước rõ ràng" |
| Data | "Như nguyên liệu, cần xử lý trước khi dùng" |
| API | "Như tờ giấy giao dịch, hai bên trao đổi thông tin" |

## Prompt Template

```
Teaching style: explain [concept] from simple to complex,
use analogies from daily life,
include specific examples,
structure with clear progression
```
