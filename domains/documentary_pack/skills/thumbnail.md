# Thumbnail Skill

> Thiết kế thumbnail cho YouTube

## Thumbnail Rules

| Rule | Value | Reason |
|------|-------|--------|
| Text on thumbnail | 1-3 words | Đọc trên mobile |
| Face/expression | Required | CTR tăng 30% |
| Contrast | High | Nổi bật trên feed |
| Colors | 2-3 max | Không rối |
| Resolution | 1280x720 | YouTube standard |

## Thumbnail Formulas

### 1. Shock + Face
```
[Shocking text] + [Expressive face] + [Contrast background]
Example: "$40M LOST" + shocked face + red background
```

### 2. Before/After
```
[Before] vs [After] + [Arrow/contrast]
Example: "POOR → RICH" + split image
```

### 3. Question
```
[Big question mark] + [Curious face] + [Topic image]
Example: "WHY?" + thinking face + relevant image
```

### 4. Number + Promise
```
[Big number] + [Promise text] + [Face]
Example: "5 SECRETS" + "They Don't Want You to Know" + face
```

## Color for Thumbnails

| Color | Use | CTR |
|-------|-----|-----|
| Red | Urgency, danger | High |
| Yellow | Attention, warning | High |
| Blue | Trust, tech | Medium |
| Green | Money, growth | Medium |

## Text on Thumbnail

| Rule | Example |
|------|---------|
| Max 3 words | ✅ "$40M LOST" |
| Big, bold font | ✅ Impact or similar |
| Outline/shadow | ✅ White text, black outline |
| Position | Center or rule of thirds |

## Quality Check

- [ ] Readable on mobile (320px width)?
- [ ] Face with expression?
- [ ] Max 3 words?
- [ ] High contrast?
- [ ] Not cluttered?
- [ ] Matches video content?
