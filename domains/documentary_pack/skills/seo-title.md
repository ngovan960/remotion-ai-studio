# SEO & Title Skill

> Tối ưu title, description, tags cho YouTube

## Title Formulas

| Formula | Example |
|---------|---------|
| [Number] + [Topic] + [Promise] | "5 AI Tools That Will Change Your Life" |
| [How to] + [Topic] + [Benefit] | "How to Make Money with AI in 2026" |
| [Question] + [Curiosity] | "Why Do 99% of People Fail at AI?" |
| [Shock] + [Topic] | "I Tried AI for 30 Days. Here's What Happened." |
| [Topic] + [Year] | "Complete AI Guide for Beginners (2026)" |

## Title Rules

| Rule | Value |
|------|-------|
| Length | 50-70 characters |
| Keyword | Front-loaded |
| Power words | Use 1-2 |
| Numbers | Use specific (5, not "several") |
| No clickbait | Honest promises |

## Power Words

| Type | Words |
|------|-------|
| Curiosity | Secret, Hidden, Unknown, Mystery |
| Urgency | Now, Today, Stop, Don't |
| Value | Free, Complete, Ultimate, Guide |
| Emotion | Shocking, Amazing, Incredible |
| Numbers | 5, 10, 100, Step-by-Step |

## Tags Strategy

| Type | Example |
|------|---------|
| Primary keyword | "AI tutorial" |
| Secondary | "artificial intelligence beginner" |
| Long-tail | "how to use AI tools for beginners 2026" |

## Quality Check

- [ ] Title under 70 characters?
- [ ] Primary keyword in first 3 words?
- [ ] Description has timestamps?
- [ ] 5-10 relevant tags?
- [ ] No misleading claims?
