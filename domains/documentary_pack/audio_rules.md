# Documentary Audio Rules

## Audio Layers

```
Layer 1: Voice-over      → 0dB (primary)
Layer 2: Music           → -15dB (background)
Layer 3: SFX             → -12dB (effects)
Layer 4: Ambience        → -20dB (atmosphere)
Layer 5: Silence         → Strategic pauses
```

## Music Rules

### Mood by Phase

| Phase | Mood | Tempo | Key |
|-------|------|-------|-----|
| Hook | Tense, building | 100-120 | Minor |
| Setup | Calm, informative | 80-100 | Major |
| Conflict | Tension, urgency | 110-130 | Minor |
| Climax | Peak, dramatic | 120-140 | Minor |
| Resolution | Peaceful, reflective | 70-90 | Major |

### Music Generation Prompts

```
Epic: "epic orchestral documentary music, cinematic, strings, 110 BPM"
Calm: "calm ambient documentary music, piano, peaceful, 85 BPM"
Tension: "tense documentary music, building strings, 120 BPM"
Mystery: "mysterious ambient, dark pads, atmospheric, 90 BPM"
Hope: "hopeful documentary music, major key, uplifting, 100 BPM"
```

## SFX Rules

| When | SFX Type | Example |
|------|----------|---------|
| Scene transition | Whoosh | "whoosh transition" |
| Data appear | Digital | "digital scan sound" |
| Reveal moment | Impact | "dramatic impact" |
| Nature scene | Ambient | "wind, birds, water" |
| Historical | Period | "horse carriage, old radio" |

## Voice-over Rules

| Phase | Speed | Pause | Tone |
|-------|-------|-------|------|
| Hook | Fast | 0.5s | Urgent |
| Setup | Medium | 1s | Informative |
| Conflict | Increasing | 0.5s | Tense |
| Climax | Fastest | 0s | Intense |
| Resolution | Slow | 2s | Reflective |

## Silence Rules

| Type | Duration | Use |
|------|----------|-----|
| Micro | 0.1-0.3s | Between sentences |
| Dramatic | 1-2s | Before reveal |
| Emphasis | 0.5-1s | After key point |
| Transition | 1-3s | Between sections |

## Quality Check

- [ ] Voice clear and audible?
- [ ] Music mood matches scene?
- [ ] SFX timing correct?
- [ ] No audio overlap?
- [ ] Silence used for emphasis?
