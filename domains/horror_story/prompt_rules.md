# Horror Story Domain Prompt Rules

## Core Principles
- Build atmosphere through suggestion, not explicit imagery
- Use sound design to create tension
- Pace reveals for maximum impact
- Let imagination fill the gaps

## Prompt Structure

### Scene Description Format
```
[SCENE_TYPE]: [HORROR_ELEMENT] + [SETTING] + [ATMOSPHERE]
Visual: [dark_eerie_visuals_with_implied_threat]
Camera: [movement_that_builds_tension]
Audio: [unsettling_sounds_and_ambience]
Duration: [seconds]
```

### Example Prompts

**Atmosphere Scene:**
```
Atmosphere: Empty hallway + Abandoned hospital + Oppressive silence
Visual: Long dark corridor, flickering lights at far end, shadows moving
Camera: Slow creep forward, slight handheld shake
Audio: Distant dripping, faint whispers, building tension
Duration: 10s
```

**Jump Scare Setup:**
```
Setup: Something behind the door + Bedroom + Anticipation
Visual: Door slightly ajar, darkness visible, nothing moving
Camera: Slow push toward door, focus pull
Audio: Heartbeat growing louder, creaking floorboards
Duration: 8s
```

**Reveal Scene:**
```
Reveal: The entity appears + Living room + Shock
Visual: Quick flash of figure, then darkness, afterimage lingers
Camera: Quick zoom, then static, slight shake
Audio: Sudden impact sound, then silence, ringing ears
Duration: 3s
```

**Aftermath Scene:**
```
Aftermath: Realization of horror + Same room + Dread
Visual: Room now empty, but something feels wrong, details changed
Camera: Slow pan showing subtle differences
Audio: Silence, then distant laughter, fading
Duration: 6s
```

## Style Guidelines

### Visual Language
- Use darkness and shadow strategically
- imply presence through absence
- Show details that shouldn't be there
- Use negative space for tension
- Color desaturation with occasional red accents

### Sound Design
- Build layers of unease
- Use silence as a weapon
- Create anticipation through repetition
- Sudden contrasts for impact
- Low frequencies for dread

### Pacing
- Slow build to quick payoff
- Let tension simmer
- Use breathing room between scares
- End on unresolved note
- Leave audience questioning

## Horror Techniques

### Atmospheric Dread
- **Setting**: Isolated, decaying, wrong
- **Lighting**: Minimal, flickering, unnatural
- **Sound**: Ambient wrongness, absence of expected sounds
- **Details**: Things slightly off, imperceptible at first

### Jump Scare Mechanics
- **Setup**: Build anticipation slowly
- **False Safety**: Moment of calm before scare
- **Payoff**: Quick, loud, visual shock
- **Aftermath**: Silence, then lingering unease

### Psychological Horror
- **Suggestion**: Imply rather than show
- **Isolation**: Character alone with threat
- **Paranoia**: Nothing can be trusted
- **Unreliable Perception**: Is it real?

### Body Horror (Subtle)
- **Implication**: Suggest rather than show
- **Sound**: Wet, organic sounds
- **Reaction**: Character's horror response
- **Aftermath**: Evidence without explanation

## Narrative Structure

### Story Arc
1. **Normal World**: Establish safety
2. **First Sign**: Something wrong
3. **Escalation**: Threat grows
4. **Confrontation**: Face the horror
5. **Resolution**: Escape or defeat
6. **Final Twist**: It's not over

### Scene Progression
- **Opening**: Establish mood
- **Build**: Increase tension
- **Peak**: Maximum fear
- **Release**: Temporary safety
- **Cliffhanger**: Unresolved ending

## Forbidden Patterns
- Don't show monsters clearly
- Avoid gore without purpose
- Don't explain everything
- Avoid predictable scares
- Don't break tension with humor
