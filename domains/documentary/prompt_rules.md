# Documentary Domain Prompt Rules

## Core Principles
- Tell factual stories with visual evidence
- Maintain objectivity and credibility
- Use visuals to support narration, not distract
- Create emotional connection through authenticity

## Prompt Structure

### Scene Description Format
```
[SCENE_TYPE]: [MAIN_SUBJECT] + [CONTEXT] + [MOOD]
Visual: [specific_visual_elements]
Camera: [movement_and_angle]
Audio: [music_and_sfx]
Duration: [seconds]
```

### Example Prompts

**Opening Scene:**
```
Establishing shot: City skyline at dawn + historical buildings + time passage
Visual: Time-lapse sunrise over colonial architecture, mist clearing
Camera: Slow aerial pan from left to right
Audio: Ambient city sounds, distant bells, subtle piano
Duration: 10s
```

**Interview Scene:**
```
Interview: Expert historian + archive room + contemplative
Visual: Medium shot of historian surrounded by artifacts, warm lighting
Camera: Static with subtle push-in
Audio: Voiceover clear, minimal background music
Duration: 12s
```

**Archive Scene:**
```
Archive: Historical photograph + black and white + nostalgic
Visual: Ken Burns effect on sepia photograph, slight grain
Camera: Slow zoom into details
Audio: Crackling film sound, period music
Duration: 8s
```

## Style Guidelines

### Visual Language
- Use concrete, specific imagery
- Avoid abstract or metaphorical visuals
- Include contextual details (dates, locations, names)
- Maintain visual consistency throughout

### Narration Support
- Visuals should illustrate spoken words
- Avoid contradicting narration
- Use text overlays for facts and statistics
- Create visual metaphors that enhance understanding

### Pacing
- Slow, deliberate scene transitions
- Allow breathing room between key points
- Use visual rhythm to match narration cadence
- Build to emotional peaks gradually

## Forbidden Patterns
- Don't use fictional or speculative imagery
- Avoid overly dramatic camera movements
- Don't mix documentary style with entertainment effects
- Avoid rapid cuts or jarring transitions
