# History Domain Prompt Rules

## Core Principles
- Present historical events with accuracy and context
- Use timelines, maps, and charts to visualize data
- Maintain period authenticity in visuals and audio
- Connect past events to present understanding

## Prompt Structure

### Scene Description Format
```
[SCENE_TYPE]: [HISTORICAL_EVENT] + [TIME_PERIOD] + [CONTEXT]
Visual: [specific_visual_elements_with_period_accuracy]
Camera: [movement_and_angle_with_era_appropriate_style]
Audio: [period_music_and_authentic_sounds]
Timeline: [optional_timeline_element]
Map: [optional_geographic_context]
Duration: [seconds]
```

### Example Prompts

**Timeline Scene:**
```
Timeline: Major events of World War II + 1939-1945 + global scale
Visual: Interactive timeline with key dates, battles, and turning points
Camera: Slow pan across timeline, zoom into significant moments
Audio: Period music, dramatic percussion for battles
Timeline: Linear with branching paths for different theaters
Duration: 15s
```

**Map Scene:**
```
Map: Roman Empire expansion + 27 BC - 476 AD + territorial growth
Visual: Animated map showing borders changing over time
Camera: Slow zoom out to show full empire, then zoom into regions
Audio: Roman-inspired music, marching sounds
Map: Political boundaries with color-coded regions
Duration: 12s
```

**Chart Scene:**
```
Chart: Population growth during Industrial Revolution + 1760-1840 + urbanization
Visual: Animated line chart showing population increase in cities
Camera: Slow pan across chart, highlight key data points
Audio: Industrial sounds, factory ambience
Chart: Line graph with animated data points appearing
Duration: 10s
```

## Style Guidelines

### Visual Language
- Use period-accurate imagery and styles
- Include contextual details (dates, locations, names)
- Maintain visual consistency within time periods
- Use color palettes appropriate to era

### Timeline Integration
- Show chronological progression clearly
- Highlight key turning points
- Use visual metaphors for time passage
- Connect related events across time

### Map Usage
- Show geographical context accurately
- Animate changes over time
- Use appropriate map styles for period
- Include relevant landmarks and boundaries

### Chart Presentation
- Present data clearly and accurately
- Use appropriate chart types for data
- Animate data reveals for engagement
- Include labels and legends

## Historical Accuracy

### Source Verification
- Cross-reference multiple sources
- Use primary sources when available
- Cite sources in documentation
- Acknowledge uncertainties

### Period Authenticity
- Use era-appropriate visuals
- Include period-accurate details
- Avoid anachronisms
- Respect cultural contexts

### Contextual Understanding
- Provide background information
- Explain significance of events
- Connect to broader historical trends
- Avoid presentism

## Forbidden Patterns
- Don't use modern imagery for historical scenes
- Avoid anachronistic language or concepts
- Don't oversimplify complex historical events
- Avoid presentism or modern judgments
- Don't mix historical periods inappropriately
